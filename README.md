# Project Codeflix - Full Cycle 3.0
This is the netflix front end for the netflix codeflix project.

## Tecnologies used
- React
- Typescript
- Tailwind
- Graphql
- Apollo
- React Query
- React Router
- NextJS
- Vercel

### How to run
- Clone the project
- Run `npm install`
- Run `json-server --watch db.json --port 3333`
- Run `npm run dev`
